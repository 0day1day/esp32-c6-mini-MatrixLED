Matrix LED

ESP32-C6 · one-click installer
  1. 1Connect your ESP32-C6 via USB to this computer (Chrome or Edge).
  2. 2Click Install below and select the serial port.
  3. 3Wait for the flash to complete, then the device reboots.
  4. 4Join the MatrixLED-Setup Wi-Fi and open 192.168.4.1 to configure your network.
Firmware served from latest release.
Requires a Chromium browser (Chrome/Edge) with WebSerial.
Source: github.com/0day1day/esp32-c6-mini-MatrixLED
